# Book Library

A simple web application to manage your personal book collection. Add, remove, and track your reading progress with this interactive library manager.

## Features

- Add new books with title, author, page count, publication year, and reading status
- Mark books as read or unread
- Remove books from your library
- Responsive design with an attractive UI
- Local storage to persist your library data

## How to Use

1. Click the "Add Book" button to open the book creation form
2. Fill in the book details (title, author, pages, publication year, and reading status)
3. Submit the form to add the book to your library
4. Use the "Read?" or "Not Read?" buttons to toggle the reading status of each book
5. Click "Remove" to delete a book from your library
